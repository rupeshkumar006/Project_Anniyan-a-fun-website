## 🎯 Overview

Project Anniyan is a fun, interactive website that brings together HTML, CSS, and JavaScript to create an engaging user experience. The project includes multiple character pages with themed content, audio clips, and styled interfaces.

## 📄 Files Description

### HTML Files
- **anniyan.html** - Main page featuring the Anniyan character with interactive elements
- **aniyan.html** - Alternate version of the Anniyan page
- **indian.html** - Page dedicated to the Indian character theme
- **maasi.html** - Page dedicated to the Maasi character theme

### Styling
- **anniyan.css** - Central stylesheet with responsive design and character-specific styling

### JavaScript
- **anniyan.js** - Interactive functionality for Anniyan pages
- **indian.js** - Interactive features for Indian character page
- **maasi.js** - Interactive features for Maasi character page

### Media Assets
- **anniyan.mp3** - Audio file for Anniyan character
- **indian.mp3** - Audio file for Indian character
- **maasi.mp3** - Audio file for Maasi character
- **indian.jpeg** - Image asset for Indian character
- **maasi.jpg** - Image asset for Maasi character

## 🚀 How to Use

1. **Clone the Repository**
   ```bash
   git clone https://github.com/rupeshkumar006/Project_Anniyan-a-fun-website.git
   cd Project_Anniyan-a-fun-website
   ```

2. **Open in Browser**
   - Simply open any `.html` file in your web browser
   - Start with `anniyan.html` for the main experience

3. **Explore Content**
   - Navigate through different character pages
   - Click on interactive elements to trigger audio and visual effects
   - Enjoy the themed content and styling
